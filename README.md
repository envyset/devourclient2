#Operation method 操作方式
forward, backward, left, right, WASD，The number keys on the right correspond to bottom left, forward down, and right down respectively.The numeric keypad on the right corresponds to the top left, up for forward movement, and up for right movement, respectively.Number key 5 activates AI.前后左右wasd，右手边数字键盘123分别对应左下，下前进和下右，右手边数字键盘789分别对应左上，上前进和上右,数字键5激活aiThis movement is not a conventional method; instead, the target is set ahead, and the AI ​​automatically finds its way.Pressing any movement key will interrupt the AI; pressing the number 5 will reactivate it.此移动并不是常规移动方式，而是设置前方为目标，ai自动寻路，若是按任意移动键则中断ai，按数字5重新激活ai。
